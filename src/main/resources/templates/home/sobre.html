HELLOBANK

Sobre

Este projeto tem como objetivo a criação de um sistema de gerenciamento de estoque, com o intuito de facilitar o controle de estoque de uma empresa.

Este projeto foi desenvolvido utilizando as seguintes tecnologias:

  • Java
  • Spring Boot
  • Thymeleaf
  • Bootstrap
  • MySQL

Para mais informações, acesse o GitHub do projeto.
